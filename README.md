# KDTree: A template class implementing a k-dimensional tree

## Overview
This project implements a K-dimentional tree data type in a C++ template class, which supports following features.

## Features
- Container-like iterator.
- Find.
- Insert.
- Erase.
- Two constructors
- Overload operator=
- ...

## Usage
In your C++ program:
```cpp
#include "KDTree.hpp"
```
